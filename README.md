IlTube
🎬 О проекте
IlTube — современная видеоплатформа, созданная с нуля. Лёгкая альтернатива YouTube, где пользователи могут загружать, смотреть и взаимодействовать с видеоконтентом. Чистый дизайн, быстрая работа, ничего лишнего.

✨ Возможности
Смотри и загружай — Делись своими видео с миром или наслаждайся контентом других
Общайся — Ставь лайки, оставляй комментарии, отвечай в обсуждениях
Организуй — Создавай плейлисты, сохраняй в избранное, следи за историей просмотров
Открывай — Смотри тренды, ищи контент, фильтруй по категориям
Настраивай — Свой профиль, страница канала, тёмная и светлая тема
Подписывайся — Следи за любимыми авторами и не пропускай новые загрузки
🎯 Почему IlTube?
Просто. Быстро. Без рекламы. Без слежки. Только видео.

Создано как душевный проект, чтобы показать, какой может быть видеоплатформа без корпоративных сложностей.
